This folder contains the arenas used in simulations.

ofstad_etal_arena.mat => arena as described in Ofstad et al. paper
nest1_drum.mat        => 2D "natural" world
nest1.mat             => 3D "natural" world
XYZ_nid1_training.mat => original data for natural world (taken from real ants' nest)
wharenas.txt          => which of these files to use in simulations

These files contain the x, y and z coordinates of objects in the world.
